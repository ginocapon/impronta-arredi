📧 Configurazione invio dal tuo hosting

Come funziona (stesso metodo di righettoimmobiliare.it):

  1. Crea il sottodominio newsletter.improntarredi.it dal pannello hosting
  2. Carica lo script PHP di invio (generato automaticamente) nel sottodominio
  3. Configura DKIM + SPF + DMARC nel DNS del dominio (anti-spam)
  4. Componi l'email qui → clicca "Copia HTML" → incolla nello script
  5. L'invio avviene in modalità warm-up graduale per non finire in spam
